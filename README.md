# 📊 E-commerce Sales Dashboard (Power BI)

## 📌 Project Overview
This project presents an **interactive Power BI dashboard** designed to analyze e-commerce sales data and generate meaningful business insights. The dashboard helps users understand overall sales performance, revenue trends, and product category performance using powerful data visualizations.

## 🎯 Key Objectives
- Analyze overall sales performance  
- Identify top-performing products and categories  
- Understand revenue trends over time  
- Enable interactive data exploration using filters and slicers  

## 🛠 Tools & Technologies
- Power BI Desktop  
- Data Cleaning & Transformation  
- Data Visualization  
- Business Intelligence  

## 📈 Dashboard Features
- Interactive Power BI dashboard  
- Visualizations such as:
  - Bar Charts
  - Line Charts
  - Pie Charts
- Filters and slicers for dynamic analysis  
- Insights on sales trends and product performance  

## 📊 Key Insights
- Identification of high-performing product categories  
- Sales trends across different time periods  
- Revenue contribution by different product segments  

## 📚 Key Learnings
- Data cleaning and preparation for analysis  
- Building interactive dashboards using Power BI  
- Creating effective visualizations for business insights  
- Understanding data-driven decision making  

## 🚀 Future Improvements
- Add more datasets for deeper analysis  
- Integrate customer segmentation insights  
- Include advanced Power BI features like DAX measures and drill-through reports
